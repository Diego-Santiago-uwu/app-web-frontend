Sitio web diseñado por Alejandra Carmen y Diego Santiago

En el frontend, la aplicación utiliza Angular para crear una interfaz de usuario interactiva y dinámica. Los componentes de Angular, como NavBarComponent, HeaderComponent, FooterComponent, ProductListComponent, DetailsProductComponent, LoginComponent y RegisterComponent, se utilizan para construir la interfaz de usuario. Además, el enrutamiento de Angular permite una navegación fluida entre diferentes vistas en la aplicación. Para el diseño y la estilización de la interfaz de usuario, se utiliza Bootstrap, y Font Awesome proporciona los iconos. Algunas interacciones de la interfaz de usuario se manejan con jQuery.

En el backend, la aplicación está basada en Java y Spring Boot, y utiliza Maven como herramienta de gestión de proyectos. La estructura del backend se organiza en torno a varias entidades principales: User, Address, ShoppingCart, Order y Product. Cada entidad tiene su propio repositorio para manejar las operaciones de base de datos, y los servicios correspondientes para implementar la lógica de negocio.

El UserService se encarga de las operaciones relacionadas con los usuarios, como la autenticación y la creación de nuevos usuarios, utilizando PasswordUtils para el manejo de contraseñas. El ShoppingCartService y OrderService manejan las operaciones relacionadas con los carritos de compras y los pedidos, respectivamente. Estos servicios son utilizados por el UserController para implementar endpoints que permiten a los usuarios agregar productos a sus carritos de compras y crear pedidos.

La aplicación utiliza Spring Security para la autenticación y la autorización, y BCrypt para el hash de las contraseñas. Los datos se almacenan en una base de datos y la aplicación utiliza Spring Data JPA para interactuar con ella.
